**Instructions for Authenticating Retailers on** [**Bol.com**](http://bol.com/)
We’re still finalizing the documentation, but if you’d like to get started, here’s how to authenticate users by generating API credentials in their [Bol.com](http://bol.com/) retailer account. This applies whether you’re using our new authenticator or token-based authentication.**Step 1: Generate API Credentials**

1. **Create API Tokens in** [**Bol.com**](http://bol.com/)**:**

   * In the [Bol.com](http://bol.com/) retailer account, navigate to **Settings > API Settings**.

2. **Contact details Setup:**

   * Fill in the company’s point-of-contact details.

3. **Create Client Credentials:**

   * Go to the **Retailer API** section.

   * Click **Create Client Credential** to generate API tokens.

4. **Copy Credentials:**

   * Copy the **Client ID** and **Client Secret**.

   * Use these credentials in our authenticator to authenticate the retailers.
